# School Management Platform

A digital platform to manage school operations in one place.

## Main Areas

- Admin dashboard
- Teachers
- Students
- Guardians
- Employees
- Library
- Bar
- Reprography
- Canteen
- Gatehouse
- Sports facilities
- Attendance
- Schedules
- Reports

## Goal

The goal of this project is to build a realistic school management system using Python, OOP, databases, dashboards, tests, Git and automation.
